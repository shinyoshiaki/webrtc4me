# webrtc4me

# NPM
https://www.npmjs.com/package/webrtc4me

`yarn add webrtc4me`  
or  
`npm i webrtc4me --save`  

# What is this?

This is a webrtc wrapper library for browser and node.js.  
Implemented very easy signaling way.  
Support multi label datachannel and media stream.
